Full SQL for Excel (ODBC, DB Updates)

This tool is part of Data Mart for Excel (D4E). Use it to prepare Excel files with SQL formulas and scheduler flows. You can design sheets in Excel for Web or Mac â€” then run Full SQL in D4E (Desktop).

To send SQL requests to D4E Server:- Click [Create Budget Application] > put your email in cell Menu!C1, then
A) Select SQL Version item (e.g., Remote GL Report), B) Or: Place cursor on compliant SQL code in any sheet (e.g., FISSQL Column D),
then click [Get Data Table from D4E Server]. This will create an email ready for the back-end server.

To set up your own D4E Server:- In Excel with Advanced D4E + Office Addin, click [Create Budget Application] > [Start Scheduler].
The server will process requests every 5 minutes and email back results. To test, download and put this file on your system: C:\temp\d4e\FinancialData.accdb
